Key Characteristics Today

  • ✅ Desktop native (Electron) — runs on user's machine
  • ✅ Multi-provider AI — LM Studio local, Groq, Cerebras, Custom
  • ✅ Browser automation — headless Chromium inside the app
  • ✅ Plugin & Skill system — user-extensible capabilities
  • ✅ Task-level self-correction — agent loop retries on error
  • ❌ No structured RSI — no audit trail, no self-improvement cycle
  • ❌ No AaaS substrate — can't deploy as a cloud agent service
  • ❌ No repo-native integration — no "clone & the agent understands"
