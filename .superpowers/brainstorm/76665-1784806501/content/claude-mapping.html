Claude pake CLAUDE.md + .claude/skills/ sebagai konfigurasi per-proyek. Mark bisa adopt pola ini: seed sources dari ~/.mark/seeds/ — user tinggal nambahin file markdown kategori.
Step 3 (Tier-1 Filter)

Step 4 (Tier-2 Semantic)
Claude pake rubrics: "Verify outcomes against a rubric using a separate grader agent. Failures loop back for rework automatically." — Mark bisa pake pola yang sama: grader agent (small model / classification) buat nge-score relevance repo.
Step 5 (Hybrid Suggestion) + Phase A
Claude's chained verification: skill A panggil skill B panggil skill C. "Members of Anthropic's Claude Code team use this pattern" — Mark bisa: RSI nemu repo → Mark cobain integrasi → Mark verify hasil → Mark lapor + register sebagai capability. Ini chained RSI loop.
